Attribute

start tag 안에는 그 element 의 부가적인 속성(attribute) 를 지정해줄수 있다

기본구문 attribute name(속성이름) = "attribute value(속성값)" 이며

여러개의 속성을 지정해줄수 있다. 띄어쓰기로 구분

속성값은 반드시 따옴표 (쌍타옴표 혹은 홀따옴표) 쌍으로 감싸주자

style 속성에 color:red 라는 속성값을 주면, 빨간글자가 된다
